# Project Proposal

An app for tracking shared household tasks.

- Google Task API https://developers.google.com/tasks/reference/rest

## Roadmap

### MVP

- Users can login
- Users can add task to list w/frequency (once, daily, weekly, monthly)
- Tasks can be assigned to user/co-user
- Users can mark tasks complete

### 2.0

- Users can visualize tasks completed, who was assigned to task
- Users can synch tasks with Google Tasks
- UI appealing

### Nice-to-haves

- React
- Pop-up tips for common tasks
